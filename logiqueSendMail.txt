2. Envoyer un vrai mail gratuitement
2.1 Méthode simple avec Gmail SMTP (gratuit mais limité)
Tu peux utiliser un compte Gmail (ton email Gmail perso ou un dédié).

Serveur SMTP : smtp.gmail.com

Port : 587 (TLS) ou 465 (SSL)

Nécessite l’authentification avec ton email + mot de passe d’application (pas ton mot de passe Gmail direct)

Important : Active la validation en deux étapes sur Gmail, crée un mot de passe d’application ici.

Exemple d’implémentation avec Spring Boot
Ajoute dans ton application.properties (ou application.yml) :

properties
Copier
Modifier
spring.mail.host=smtp.gmail.com
spring.mail.port=587
spring.mail.username=ton.email@gmail.com
spring.mail.password=ton_mot_de_passe_app
spring.mail.properties.mail.smtp.auth=true
spring.mail.properties.mail.smtp.starttls.enable=true
Crée un service mail simple :

java
Copier
Modifier
import org.springframework.mail.SimpleMailMessage;
import org.springframework.mail.javamail.JavaMailSender;
import org.springframework.stereotype.Service;

@Service
public class EmailService {

    private final JavaMailSender mailSender;

    public EmailService(JavaMailSender mailSender) {
        this.mailSender = mailSender;
    }

    public void sendSimpleEmail(String to, String subject, String text) {
        SimpleMailMessage message = new SimpleMailMessage();
        message.setFrom("ton.email@gmail.com");
        message.setTo(to);
        message.setSubject(subject);
        message.setText(text);
        mailSender.send(message);
    }
}
Dans ton controller, injecte ce service et remplace le System.out.println par un appel à emailService.sendSimpleEmail(...).

2.2 Autres solutions gratuites
SendGrid (offre gratuite : 100 mails/jour)

Crée un compte sur sendgrid.com

Récupère une API Key

Utilise la lib Java officielle SendGrid ou envoie via SMTP SendGrid (comme Gmail)

Mailjet (offre gratuite similaire)

Amazon SES (plus complexe mais possible)

Résumé :
Pour un test rapide : utilise Gmail SMTP

Pour envoyer en prod : pense à SendGrid ou Mailjet (plus pro, plus fiable)
